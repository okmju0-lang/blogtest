## 이미지 배치 원칙

1. **첫 번째 H2 섹션 아래**: 독자의 시선을 잡는 핵심 비주얼
2. **핵심 논점 섹션**: 추상적 개념을 시각화하는 일러스트
3. **사례/도구 언급 부분**: 관련 스크린샷
4. **구조/프로세스 설명**: 다이어그램
5. **마지막 섹션 전**: 정리/요약 성격의 비주얼

## 스크린샷 캡처 대상 우선순위

1. 본문에 직접 언급된 서비스/도구의 공식 페이지
2. 본문에 언급된 오픈소스 프로젝트의 GitHub 페이지
3. 관련 기술 블로그나 공식 문서
4. 산업 보고서나 통계 페이지
