# 기밀 정보 필터링 체크리스트

## 목적

블로그 최종 발행 전 내부 기밀 정보가 노출되지 않도록 검증한다.

---

## 체크리스트

### 1. 재무 정보

| 항목 | 판정 기준 | 처리 방법 |
|---|---|---|
| 매출/수익 수치 | 공개 발표된 수치가 아닌 경우 | 삭제 또는 "공개 예정" 처리 |
| 계약 금액 | 계약서, 입찰 금액 등 | 삭제 |
| 투자 유치 금액 | 공식 발표 전 | 삭제 |
| 비용 데이터 | 내부 원가, 마진율 등 | 삭제 |
| 직원 급여/처우 | 모든 경우 | 삭제 |

### 2. 고객사 정보

| 항목 | 판정 기준 | 처리 방법 |
|---|---|---|
| 고객사명 | 브리핑에 "비공개" 또는 "NDA" 표시 | 익명화 (예: "국내 대형 제조사") |
| 고객사 내부 정보 | 고객 동의 없이 공개된 정보 | 삭제 또는 일반화 |
| 프로젝트명 | 비공개로 표시된 경우 | 익명화 또는 삭제 |
| 사용 기술/시스템 | 고객사 기밀 인프라 정보 | 삭제 |

### 3. 내부 전략/로드맵

| 항목 | 판정 기준 | 처리 방법 |
|---|---|---|
| 미발표 신제품/서비스 | 발표 전 | 삭제 |
| 사업 확장 계획 | 공식 발표 전 | 삭제 또는 일반화 |
| 기술 로드맵 | 내부 전략 문서 내용 | 삭제 |
| 파트너십/인수합병 | 공개 전 | 삭제 |
| 조직 개편 계획 | 공개 전 | 삭제 |

### 4. 개인정보

| 항목 | 판정 기준 | 처리 방법 |
|---|---|---|
| 직원 이메일 | 모든 개인 이메일 | 삭제 (대표 이메일만 허용) |
| 전화번호 | 개인 연락처 | 삭제 |
| 주민등록번호 | 모든 경우 | 즉시 삭제 |
| 직원 개인 정보 | 동의 없이 공개 | 삭제 또는 익명화 |
| 고객 개인 정보 | 모든 경우 | 삭제 |

## 판정 가이드

### 확실히 기밀 (confidential)
- 브리핑에 "비공개", "NDA", "내부용", "대외비" 등으로 명시된 항목
- 수치 데이터 중 공개 발표된 적이 없는 것
- 개인정보 패턴 탐지기가 검출한 항목

### 무해 (safe)
- 이미 공개 발표된 정보 (보도자료, 공식 사이트 등)
- 익명화/일반화된 정보
- 회사 대표 연락처 (공식 채널)

### 불명확 (escalate)
- 위 기준으로 판단하기 어려운 경우
- 브리핑에 공개 여부가 명시되지 않은 구체적 수치
- 고객사명이 언급되었으나 공개 여부 불분명

> **불명확한 경우 원칙**: 삭제/익명화를 기본으로 하고, 담당자에게 에스컬레이션한다.

---

## 에스컬레이션 조건

다음 중 하나라도 해당하면 즉시 담당자에게 보고한다:

1. 주민등록번호, 신용카드 번호 등 민감한 개인정보 발견
2. NDA 대상 고객사 실명 노출
3. 미공개 재무 수치 (매출, 계약 금액 등)
4. 공개 전 사업 계획 또는 기술 로드맵
5. 판단이 모호한 항목이 3개 이상인 경우
