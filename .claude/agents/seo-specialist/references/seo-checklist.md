# SEO 체크리스트 (한국어 블로그 범용)

## 1. 제목 (Title Tag)

- [ ] 타겟 키워드가 제목 앞부분에 위치
- [ ] 제목 길이: 30~60자 (한글 기준)
- [ ] 클릭 유도 요소 포함 (숫자, 질문형, 혜택 명시)
- [ ] 중복 키워드 나열 금지 (키워드 스터핑)

**좋은 예**: `GPT-4o로 바뀐 업무 방식: AX 담당자가 알아야 할 3가지`
**나쁜 예**: `AI AI트렌드 GPT 인공지능 최신 트렌드 2026`

---

## 2. 메타 디스크립션 (Meta Description)

- [ ] 길이: 80~120자 (한글 기준, 검색결과 표시 기준)
- [ ] 타겟 키워드 자연스럽게 포함
- [ ] 독자가 얻을 혜택 또는 클릭 이유 명시
- [ ] 행동 유도 문구 포함 (예: "확인해보세요", "알아보세요")

---

## 3. 타겟 키워드

- [ ] 메인 키워드 1개 선정 (검색량 고려)
- [ ] 서브 키워드 2~3개 선정
- [ ] 메인 키워드 배치: 제목, 첫 문단, H2 소제목 중 1개 이상, 본문 자연스럽게
- [ ] 키워드 밀도: 본문의 1~2% 수준 (과최적화 금지)

**한국어 키워드 선정 팁**:
- 영어 → 한국어 번역 키워드 vs 영어 원문 키워드 모두 고려
- 예: "AI Transformation" vs "AI 전환" vs "인공지능 혁신"
- 롱테일 키워드 활용 (예: "중소기업 AI 도입 방법")

---

## 4. 헤딩 구조 (H1/H2/H3)

- [ ] H1: 글 제목 1개만 사용
- [ ] H2: 주요 섹션 구분 (3~6개 권장)
- [ ] H2/H3: 타겟 키워드 또는 관련 키워드 자연스럽게 포함
- [ ] 헤딩 계층 구조 준수 (H2 없이 H3 사용 금지)

---

## 5. 본문 콘텐츠

- [ ] 글 길이: 최소 1,000자 (권장 2,000자 이상)
- [ ] 첫 문단: 핵심 메시지 포함, 타겟 키워드 자연스럽게 등장
- [ ] 단락 구분: 3~5문장 단위 (가독성)
- [ ] 리스트/표 활용: 정보 정리 가독성 향상

---

## 6. 이미지 SEO

- [ ] 모든 이미지에 alt 텍스트 작성
- [ ] alt 텍스트: 이미지 내용 설명 + 가능하면 키워드 포함
- [ ] 파일명: 의미 있는 이름 사용 (예: `ax-transformation-process.png`)

---

## 7. 내부 링크

- [ ] 관련 기존 블로그 글 2~3개 연결 (가능한 경우)
- [ ] 앵커 텍스트: 링크 대상 글의 키워드 포함
- [ ] "여기를 클릭하세요" 같은 비서술적 앵커 텍스트 금지

---

## 8. URL 구조 (참고)

> 블로그 플랫폼이 결정되면 업데이트 필요

- 한글 URL 가능 여부 확인
- 영문 URL 권장: 짧고 키워드 포함 (예: `/ai-trend-2026`)
- 특수문자, 날짜 과다 포함 지양

---

## 카테고리별 SEO 포인트

| 카테고리 | 주요 키워드 방향 | 특이사항 |
|---|---|---|
| Case Study | "[산업] AI 도입 사례", "[문제] 해결 방법" | 성과 수치를 제목/메타에 포함하면 CTR 향상 |
| Thought Leadership | "AX 전략", "[직군] AI 활용" | 검색보다 소셜 공유 중심 최적화 |
| Company News | 회사명 + 이벤트 키워드 | 브랜드 검색 최적화 우선 |
| AI Trend | 트렌드 키워드 + "의미", "영향", "방법" | 검색량 높은 시사성 키워드 우선 |
