## 3줄 요약

1. Microsoft Research 논문으로, 기존 RAG가 "데이터셋 전체의 주요 테마는?" 같은 글로벌 질문에 실패하는 문제를 해결하기 위해 Graph RAG를 제안한다.
2. 소스 문서에서 엔티티 지식 그래프를 도출한 뒤, 밀접 관련 엔티티 그룹의 커뮤니티 요약을 사전 생성하는 2단계 그래프 인덱스를 구축한다.
3. 100만 토큰 규모 데이터셋에서 기존 RAG 대비 답변의 포괄성(comprehensiveness)과 다양성(diversity)이 크게 향상되었다.

## 핵심 인사이트

1. **QFS 문제로의 재정의**: 글로벌 질문은 검색(retrieval)이 아닌 쿼리 중심 요약(Query-Focused Summarization) 문제임을 명확히 했다.
2. **커뮤니티 요약 → 부분 응답 → 최종 합성**: 각 커뮤니티 요약으로 부분 응답을 생성하고 이를 다시 합산하는 map-reduce 방식이 핵심이다.
3. **확장성 확보**: 사용자 질문의 일반성과 소스 텍스트 양 모두에 대해 확장 가능한 구조를 제시한다.

## AX 관련성

- **태그**: 기술트렌드, 데이터활용
- **관련성 설명**: 대규모 사내 문서에서 전체적 인사이트를 도출하는 능력은 AX 전략 수립 시 데이터 기반 의사결정의 핵심 역량이 된다.
